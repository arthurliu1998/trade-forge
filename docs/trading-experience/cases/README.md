# Trade Case Studies

## Naming Convention

Files are named: `case-NNN-short-title.md`

Examples:
- `case-001-aapl-rsi-oversold-bear.md`
- `case-002-2330-foreign-flow-reversal.md`
- `case-003-nvda-earnings-gap.md`

## Template

Use `case-000-template.md` as the starting point for new cases.
